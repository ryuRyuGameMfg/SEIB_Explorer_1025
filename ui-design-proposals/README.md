# SEIB Explorer UI Design - Modern SF

SEIB ExplorerのUI改善デザイン（最終版）

## 🎨 デザインコンセプト

**Modern SF（モダン・SF）** - 未来的でモダンなSFスタイル

### 主な特徴

- ✨ **ダークブルー・パープルのグラデーション配色**
- 🎯 **8つのパネル完全対応**（タブ切り替え）
- 💫 **グロー効果とアニメーション**
- 📐 **16:9固定アスペクト比**
- 🇯🇵 **完全日本語対応**
- 🖱️ **直感的なボタン操作**（16:9コンテナの外に配置）

---

## 🚀 使い方

ブラウザでHTMLファイルを開いてください：

```bash
open modern-sf/index.html
```

---

## 📊 8つのパネル

画面上部のボタンで切り替え可能：

| アイコン | パネル名 | 内容 |
|---------|---------|------|
| 🌍 | シミュレーション情報 | 地点情報・土壌パラメータ・CO₂グラフ |
| 📊 | 凡例 | 植物機能型（PFT）の色分け |
| 🔄 | 物質循環 | 年間フラックス図 |
| 🌿 | 炭素循環 | 炭素ストック・外部交換 |
| 💧 | 水循環 | 水フラックス・積雪動態 |
| 🌤️ | 大気環境 | 気温・風速・湿度 |
| ☀️ | 放射収支 | 短波・長波放射 |
| 🌲 | 植物特性 | バイオマス構成・森林構造 |

---

## 🎮 操作方法

### パネル切り替え
- **マウスクリック**: 上部のボタンをクリック
- アクティブなボタンは**明るく光り、浮き上がります**

### ビジュアル効果
- ボタンホバー：明るくなり上昇
- アクティブ時：グラデーション背景＋グロー効果＋パルスアニメーション
- パネル切り替え：スムーズなフェードイン

## 📝 技術仕様

- **アスペクト比**: 16:9固定
- **レスポンシブ**: 画面サイズに応じて自動スケール
- **ブラウザ対応**: モダンブラウザ（Chrome, Firefox, Safari, Edge）
- **依存関係**: なし（純粋なHTML/CSS/JavaScript）

---

## 💡 Unity実装に向けて

このプロトタイプは、UnityのUIシステムで再現可能です：

- **パネル切り替え**: UnityのCanvas Group を使用
- **ボタン**: Unity UI Button コンポーネント
- **アニメーション**: DOTween または Animator
- **グラフ**: Graph And Chart アセット（既存）
- **配色**: カラーパレットをScriptableObjectで管理
